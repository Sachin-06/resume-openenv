## Environment Overview
This environment simulates a recruiter evaluating candidates based on skills and experience.

## Action Space
- shortlist
- reject

## Observation Space
- candidate_skills
- experience
- job_required_skills

## Tasks
- Easy: Perfect skill match
- Medium: Partial match
- Hard: Misleading candidate

## Baseline Result
The baseline agent achieves a score of **1.0** across all tasks.
